# Minecraft Mod Installer

> ## Fonctionnement du programe :

### Quand vous ouvrez le programe, il va vous demander votre loader (Fabric,Forge,Neo forge,ect...), pui après avoir séléctioné votre loader, un dossier sera créer pour mettre les mod que vous voulez installer. Relancer le programe et reséléctioner le loader pui vos mod seron automatiquement ajouter !

> ### Atention ! Le programe n'installe pas encore le loader automatiquement !
